Bạn đang tìm cách tạo một trang portfolio cá nhân để gây ấn tượng với nhà tuyển dụng? Trong bài viết này, mình sẽ hướng dẫn các bạn tạo một portfolio website hoàn chỉnh chỉ trong 30 phút!

🎯 Mục tiêu của bài viết #

Sau khi hoàn thành bài viết này, bạn sẽ có:

  • Một portfolio website responsive, đẹp mắt
  • Hiểu cách sử dụng React Hooks
  • Biết cách style với Tailwind CSS
  • Deploy website lên Vercel hoàn toàn miễn phí

📋 Yêu cầu trước khi bắt đầu #

  • Node.js đã được cài đặt (version 16+)
  • Biết JavaScript cơ bản
  • Có tài khoản GitHub và Vercel

🚀 Bước 1: Khởi tạo dự án #

Mở terminal và chạy lệnh:

npx create-react-app my-portfolio
cd my-portfolio
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

⚙️ Bước 2: Cấu hình Tailwind CSS #

Mở file tailwind.config.js và thay thế nội dung:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        primary: '#3b82f6',
        secondary: '#8b5cf6',
      }
    },
  },
  plugins: [],
}

Trong file src/index.css, thêm:

@tailwind base;
@tailwind components;
@tailwind utilities;

🚢 Bước 6: Deploy lên Vercel #

  1. Push code lên GitHub
  2. Truy cập vercel.com
  3. Import repository
  4. Click Deploy!

Chỉ 3 bước và website của bạn đã live! 🎉
