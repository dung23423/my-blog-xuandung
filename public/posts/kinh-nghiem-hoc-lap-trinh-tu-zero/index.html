Cách đây 1 năm, mình chỉ là một sinh viên năm 2 không biết gì về lập trình ngoài HTML cơ bản. Hôm nay, mình đang làm Junior Developer với mức lương 10 triệu/tháng. Đây là câu chuyện của mình.

🎬 Bắt đầu từ đâu? #

Tháng 10/2024, mình quyết định nghiêm túc với lập trình sau khi xem một video về “A day in life of a Developer”. Lúc đó mình:

  • Không biết JavaScript
  • Chưa từng dùng Git
  • Không hiểu Frontend/Backend là gì

Nghe có vẻ tệ? Nhưng đó là xuất phát điểm của hầu hết mọi người!

📚 Lộ trình học của mình #

Tháng 1-2: Nền tảng (2-3h/ngày) #

HTML & CSS

  • Học từ W3Schools
  • Làm 5 landing page mẫu
  • Clone trang chủ của Facebook, Google

JavaScript cơ bản

  • Course: JavaScript for Beginners (FreeCodeCamp)
  • Project: Máy tính đơn giản
  • Project: Todo App
  • Project: Weather App

Sai lầm đã mắc phải: Học quá nhiều lý thuyết, ít code. Nên làm project ngay từ đầu!

Tháng 3-4: Framework đầu tiên (3-4h/ngày) #

React.js

  • Tutorial chính thức của React
  • Làm lại 3 project trước bằng React
  • Học React Hooks, Context API
  • Project lớn: Blog cá nhân với Firebase

Tools học được:

  • Git & GitHub
  • VS Code + Extensions
  • Chrome DevTools
  • NPM & Package management

Tháng 5-6: Backend & Database (4h/ngày) #

Node.js & Express

  • REST API cơ bản
  • Authentication với JWT
  • CRUD operations

Database

  • MongoDB với Mongoose
  • PostgreSQL cơ bản
  • Firebase Firestore

Project thực chiến:

  • E-commerce API
  • Social Media API
  • Chat Application (Socket.io)

Tháng 7-8: Full-stack Projects #

Làm 3 dự án full-stack hoàn chỉnh:

  1. Task Management App

    • Frontend: React + Tailwind
    • Backend: Node.js + Express
    • DB: MongoDB
    • Deploy: Heroku + Netlify

  2. Blog Platform

    • Next.js + MDX
    • CMS: Contentful
    • Deploy: Vercel

  3. E-commerce Website

    • React + Redux
    • Node.js + PostgreSQL
    • Payment: Stripe
    • Deploy: Railway + Vercel

Tháng 9-10: Chuẩn bị xin việc #

  • Hoàn thiện Portfolio
  • Viết CV chuyên nghiệp
  • Luyện thuật toán trên LeetCode (Easy)
  • Mock interview
  • Apply 50+ công ty

💼 Kết quả #

Tháng 10/2025:

  • Nhận 8 lời mời phỏng vấn
  • Pass 3 công ty
  • Chọn công ty startup với:
    • Lương: 10 triệu/tháng
    • Review lương sau 3 tháng
    • Được mentor 1-1
    • Làm việc hybrid

🎯 Những điều quan trọng nhất #

1. Consistency > Intensity #

Học 2h/ngày đều đặn tốt hơn 10h/ngày rồi nghỉ 1 tuần.

2. Build, Build, Build #

Code > Tutorial. Đừng rơi vào “Tutorial Hell”.

3. Document your journey #

  • GitHub profile đều đặn
  • Viết blog về những gì học được
  • Chia sẻ trên LinkedIn

4. Community matters #

  • Join Discord/Slack communities
  • Tham gia meetup
  • Contribute Open Source

5. Don’t compare #

Người khác 3 tháng xong, bạn 1 năm cũng OK. Mỗi người một hành trình.

❌ Những sai lầm cần tránh #

  1. Tutorial Hell - Học mãi không làm
  2. Học quá nhiều thứ cùng lúc - Focus vào 1 stack
  3. Không làm project - Project = Portfolio
  4. Sợ apply việc - Apply sớm để biết mình thiếu gì
  5. Không học English - 80% tài liệu tốt là tiếng Anh

💰 Chi phí #

  • Courses: ~2 triệu VND
  • Domain & Hosting: ~500k VND/năm
  • Coffee ☕: Vô số 😅

Total: Khoảng 3 triệu cho cả năm

🚀 Kế hoạch tiếp theo #

  • Học TypeScript
  • Deep dive vào System Design
  • Contribute nhiều hơn cho Open Source
  • Target: Mid-level trong 1 năm nữa

💬 Lời khuyên cuối cùng #

  • ❌ Bằng CS
  • ❌ Toán giỏi
  • ❌ IQ cao

Chỉ cần:

  • ✅ Kiên trì
  • ✅ Học mỗi ngày
  • ✅ Build projects
  • ✅ Never give up
